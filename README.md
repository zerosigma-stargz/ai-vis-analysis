# gen-graphics-modif-df
Aplikasi ini ai chatbot analisis data dan auto visualisasi grafik berbasis web yang dibangun dengan Python dan framework Streamlit. Pengguna mengunggah file data berbagai format, melihat ringkasan dan statistik data otomatis, lalu interaksi antarmuka chat berbasis Gemini AI melakukan transformasi DataFrame dan hasil berbagai grafik/chart dinamis.
